# api-client-jsonplaceholder-typicode-com

Auto-generated API client SDK.

## Installation

```bash
go get api-client-jsonplaceholder-typicode-com
```

## Usage

```go
import "api-client-jsonplaceholder-typicode-com"

client := api-client-jsonplaceholder-typicode-com.NewApiClient("", nil)
data, err := client.GetData()
```
